  Next steps on the Pi:                                                                                                       
  # 1. Add the LED submodule                                                                                                  
  git submodule add https://github.com/jgarff/rpi_ws281x.git third_party/rpi_ws281x
                                                                                                                              
  # 2. Install deps                                                                                                           
  sudo apt install -y build-essential cmake pkg-config libpulse-dev libfftw3-dev                                              
                                                                                                                              
  # 3. Build                                                                                                                  
  cmake -B build -DCMAKE_BUILD_TYPE=Release                 
  cmake --build build -j$(nproc)                                                                                              
                                                                                                                              
  # 4. Run (needs root or cap_sys_rawio for DMA)
  sudo build/musevis musevis_sink.monitor                                                                                     
                                                            
  GPIO note: If using the Pi's 3.5mm audio jack, change GPIO_PIN in LEDController.cpp from 18 to 10 and configure rpi_ws281x
